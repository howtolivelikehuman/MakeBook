# MakeBook
책 만들기 어플리케이션

## 기능
● 내가 원하는 E북 만들기

● 원하는 표지를 그려서 만들기

● 직접 글을 작성하고, 이미지를 삽입하기

● 원하는 페이지를 직접 읽어 녹음하기

● 내가 만든 책들을 한 눈에 확인하기

● 만든 책을 PDF 문서로 변환하기
